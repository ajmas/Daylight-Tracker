Daylight Tracker

This applet is a port to Java of the 'night & day' algorithm from XsunClock, by John Walker. The port was performed by André-John Mas. The source code is available upon request - my e-mail address is available on my web page.

Compatibity issues: For some reason the night does not get drawn in some version of Internet Explorer. This may be due to Microsoft's version of Java. Installing Sun's Jave Runtime Environment, should solve the issue. I am currently investegating this.
